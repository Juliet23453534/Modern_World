Meaning Of "Computer Basics"

What is "Computer Basics"? It basically means how to use computer and how it works. It is very urgent because you cannot use a computer properly if you don't know its basics. On this web page, we will show you some main components of computers and also shortcut keys.




Computer Components

Computer Mouse

Computer mouse is to control the computer and it can be wireless or with the wire.

Hard Drive

Hard drives are data storage devices used to store files, programs or other information on a computer system.

Cooling Fan

Cooling fan can decrease overheating. If you play heavy game cooling it will be so hot so you need to use cooling.

RAM

RAM is a common computing acronym that stands for random-access memory. RAM allows your computer to perform most of its everyday tasks, such as loading applications, browsing the internet, editing a spreadsheet, or experiencing the latest game.

CPU

The CPU is the most important part of the computer. When the CPU has a problem, a computer technician may start by checking the fan and cleaning out.

Monitor

A computer monitor is an electronic device that displays what's what programs are running on your computer so a user can see.

Short Cut Keys

ShortCut Keys Description
Alt+F File menu options in the current program.
Alt+E Open Edit options in the current program.
Alt+Tab Switch between open programs.
F1 View help information (F1 is used by almost every Windows program to display help).
F2 Rename a selected file.
F5 Refresh the current program window.
Ctrl+D Bookmark the current page in most Internet browsers.
Ctrl+N Create a new or blank document in some software or open a new tab in most Internet browsers.
Ctrl+S Save the current document file.
Ctrl+Z Undo the last action.
Ctrl+C Copy the selected item.
Ctrl+V Paste
Ctrl+Esc Open the Start menu.
Alt+Enter Open the properties for the selected item (file, folder, shortcut, etc.).
Alt+F4 Close the currently active program.
Window Key+L Lock the computer, requiring password entry to access again.
